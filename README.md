# This Project

This project is a simple React App that uses API calls to fetch data and display it on the page. It also includes a feature to add new data and to delete existing data.

The app is built with React, React Router, and Axios. It uses CSS for styling and JavaScript for logic.

The app is deployed on Vercel and can be accessed at [https://react-app-omega.vercel.app/](https://react-app-omega.vercel.app/).

## Features

* The app displays a list of characters from the One Piece API.
* The app allows users to add new characters to the list.
* The app allows users to delete characters from the list.

## Installation

To install the app, clone the repository and run `npm install` in the root directory.

## Deployment

To deploy the app, run `npm run build` and then `npm run deploy`. The app will be deployed to Vercel.

## Development

To develop the app, run `npm start` in the root directory. The app will be served at [http://localhost:3000](http://localhost:3000).

## Technologies Used

* React
* React Router
* Axios
* CSS
