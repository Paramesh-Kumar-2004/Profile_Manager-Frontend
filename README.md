# Student Management System

This is a React based project for creating a Student Management System. The system is designed to manage students, teachers, and classes. It allows users to create, read, update, and delete students, teachers, and classes. The system also allows users to assign teachers to classes and students to classes.

## Features

- Create, read, update, and delete students, teachers, and classes
- Assign teachers to classes and students to classes
- Search for students, teachers, and classes

## Technology Used

- React
- React Hooks
- React Router
- React Context API
- React Toastify
- Axios
- Day.js
- Change Case
- React-Toastify
- React-Router-Dom
- React-Refresh
- React-Loader

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Usage

1. Go to `http://localhost:3000`
2. Click on the links to navigate to the different pages
3. Use the search bar to search for students, teachers, and classes
4. Use the buttons to create, read, update, and delete students, teachers, and classes
5. Use the dropdowns to assign teachers to classes and students to classes
